# 📚 Assignment Tracker

## What This Software Does
Assignment Tracker is a Java program that helps students organize homework assignments and due dates.

Users can:
- Add assignments
- View assignments in a table
- Search assignments by subject
- Remove completed assignments

---

## Who It’s For
This program is designed for students who want an easy way to keep track of homework and school assignments, especially AP students.

---

## How to Run the Program
1. Open the project in a Java IDE
2. Compile all Java files
3. Run `Main.java`
4. Follow the console prompts

---

## Technical Overview

### Main Classes
- `Main`
  Handles user input and runs the program.

- `Assignment`
  Stores assignment name, subject, and due date.

- `AssignmentManager`
  Manages assignments using an `ArrayList`.

### Key Features
- Uses objects and classes
- Uses an `ArrayList`
- Uses a 2D array to display assignment data
- Uses loops, methods, and conditionals
